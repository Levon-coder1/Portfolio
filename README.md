# Divinelove Portfolio

Modern, Apple-inspired single-page portfolio for Divinelove — software engineer and gadget vendor. The page blends cinematic gradients, layered glass, and parallax micro-interactions similar to webenart.com’s flow.

## Getting started

Open `index.html` in your browser to explore the experience. All assets are self-contained (HTML/CSS/JS only).

## Highlights
- Multi-section flow with hero, work showcase, services, device drops, and CTA.
- Parallax tags, tilt-on-hover cards, marquee stack strip, and reveal-on-scroll animations.
- Gradient-heavy, glassmorphism aesthetic tuned for desktop and mobile.
